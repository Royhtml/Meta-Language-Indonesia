## [1.0.0] - Initial Release
### Added
- Initial release of the Meta Language Indonesia extension.
- Basic grammar and syntax highlighting support for `.meta` files.
- *On-the-fly* transpiler to convert Meta syntax (Indonesian) to Python.
- Core language conversion features: loops, conditionals, and built-in functions (`cetak`, `masukan`, etc.).
